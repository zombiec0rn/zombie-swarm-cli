# Zombie Swarm Cli

Command line interface to [zombie swarm]().

Used to query and manipulate zombie swarms.

## Install

```sh
npm install -g @zombiec0rn/zombie-swarm-cli
```

## Use

```
zombie-swarm ls --interface eth0
```

## Options

```
ls  - list swarm nodes
plan - create a plan
apply - apply a plan
```
